# FixBattery

Landing page para el servicio de cambio de baterías de iPhone con repuestos Ampsentrix.

## Requisitos previos

Node.js

## Correr en local

1. Instalar dependencias:
   `npm install`
2. Configurar las variables de entorno necesarias en un archivo `.env` (ver `.env.example`)
3. Iniciar la app:
   `npm run dev`

## Build de producción

```
npm run build
npm start
```
